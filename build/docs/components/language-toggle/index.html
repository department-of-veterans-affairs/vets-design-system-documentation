Usage

When to use

  • Offering access to translated versions of a page
  • Users need to switch between corresponding pages in different languages
  • Content is available in multiple VA.gov-supported languages (English, Spanish, Tagalog)

When to consider something else

  • Linking to unrelated pages in different languages
  • Adding or removing languages beyond the three supported by VA.gov
  • Providing language options beyond English, Spanish, and Tagalog

Code usage

Usage
<va-language-toggle
  en-href="/health-care/covid-19-vaccine/"
  es-href="/health-care/covid-19-vaccine-esp/"
  tl-href="/health-care/covid-19-vaccine-tag/"
></va-language-toggle>

Content considerations

  • Ensure all linked pages have corresponding translated content
  • Maintain consistent URL patterns across languages
  • Link text should appear in the native language (e.g., "Espa√±ol" not "Spanish")

Accessibility considerations

  • Set the lang attribute on the article/main element to match page language
  • Label links with appropriate language codes
  • Link text should appear in the native language for proper pronunciation by screen readers
  • Ensure the toggle is keyboard accessible
