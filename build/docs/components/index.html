Components

Components are interactive building blocks of the VA.gov Design System. They are reusable chunks of code used to create the user interface of VA.gov. Teams at VA.gov use design system components to ensure a consistent user experience for Veterans and their families.

USWDS v3 Migration

Many components in this library have been updated to align with the U.S. Web Design System (USWDS) v3 while maintaining VA-specific design elements and accessibility requirements.

Component governance​

Required usage​

All teams building on VA.gov must use v3 components exclusively. Using a v1 component will be considered a launch-blocking issue.

No imposter components​

Teams must not create custom-coded elements that mimic design system components. This is a design system violation. Exceptions include:

  • Basic HTML <button> elements styled as the default button variation
  • Basic HTML <a> tags styled as default links

Components vs. Templates vs. Patterns​

ConceptDescriptionExample
ComponentsSmallest UI building blocksButton, Alert, Card
TemplatesSingle-page layouts combining componentsForm introduction page
PatternsMulti-page solutions for user needsApply for benefits flow

Component categories​

Feedback & Alerts​

Inform users about important information, status changes, and system messages.

  • Alert - Important and time-sensitive information
  • Alert - Expandable - Collapsible important information
  • Alert - Sign-in - Authentication-related alerts
  • Banner - Site-wide announcements
  • Loading indicator - System processing feedback
  • Modal - Focused interactions requiring user response
  • Snackbar - Brief feedback messages

Navigation​

Help users move through the site and understand their location.

  • Back to top - Return to page top
  • Breadcrumbs - Location within site hierarchy
  • Footer - Site-wide footer navigation
  • Header - Site-wide header navigation
  • Link - Navigation between pages
  • On this page - Jump links within a page
  • Pagination - Navigate multi-page content
  • Side navigation - Section navigation
  • Tabs - Switch between related content views

Actions​

Enable users to take action and interact with the interface.

  • Button - Primary and secondary actions
  • Button group - Multiple related buttons
  • Button - Icon - Tertiary actions with icons
  • Button - Segmented - Toggle between options
  • Critical Action - High-stakes confirmations
  • Link - Action - Prominent call-to-action links

Form Components​

Collect information from users through forms.

  • Form overview - Form component guidance and patterns
  • Autosave - Save progress indicator
  • Checkbox - Multi-select options
  • Combo box - Searchable dropdown
  • Date input - Date selection with dropdowns
  • File input - File uploads
  • Label - Form field labels
  • Memorable date - Date entry with text fields
  • Need help? - Contact information footer
  • Penalty notice - Legal disclosure
  • Prefill - Pre-populated data notice
  • Privacy Agreement - Privacy policy acceptance
  • Progress bar - Segmented - Multi-step form progress
  • Radio button - Single-select options
  • Select - Dropdown selection
  • Statement of truth - Digital signature
  • Telephone input - Phone number entry
  • Text input - Single-line text entry
  • Textarea - Multi-line text entry

Content Display​

Present information and content to users.

  • Accordion - Expandable content sections
  • Additional info - Supplementary information toggle
  • Card - Grouped content container
  • Divider - Visual content separator
  • Icon - Visual symbols and indicators
  • Link - Collection - Related links grouped together
  • List - Ordered and unordered content lists
  • OMB info - OMB control number display
  • Process list - Step-by-step instructions
  • Progress bar - Activity - Upload/processing progress
  • Summary box - Key information highlight
  • Table - Tabular data display
  • Tag - Labels and categories
  • Telephone - Phone number display
  • Text - Typography and text styling

Specialized Components​

VA-specific components for unique use cases.

  • Address block - Formatted address display
  • Crisis Line Modal - Veterans Crisis Line
  • Language toggle - Language selection
  • Search Filter - Filter search results
  • Search Input - Search functionality
  • Service List Item - Service offerings display

Deprecated components​

The following components are deprecated and should not be used in new development:

  • Banner - Promo - Use standard Banner instead
  • Input message - Use Alert (slim variant) instead
  • Notification - Use Alert instead
