# Components

Components are interactive building blocks of the VA.gov Design System. They are reusable chunks of code used to create the user interface of VA.gov. Teams at VA.gov use design system components to ensure a consistent user experience for Veterans and their families.

:::info USWDS v3 Migration
Many components in this library have been updated to align with the U.S. Web Design System (USWDS) v3 while maintaining VA-specific design elements and accessibility requirements.
:::

---

## Component governance

### Required usage

All teams building on VA.gov **must use v3 components exclusively**. Using a v1 component will be considered a launch-blocking issue.

### No imposter components

Teams must not create custom-coded elements that mimic design system components. This is a design system violation. Exceptions include:
- Basic HTML `<button>` elements styled as the default button variation
- Basic HTML `<a>` tags styled as default links

### Components vs. Templates vs. Patterns

| Concept | Description | Example |
|---------|-------------|---------|
| **Components** | Smallest UI building blocks | Button, Alert, Card |
| **Templates** | Single-page layouts combining components | Form introduction page |
| **Patterns** | Multi-page solutions for user needs | Apply for benefits flow |

## Component categories

### Feedback & Alerts
Inform users about important information, status changes, and system messages.

- [Alert](./alert/) - Important and time-sensitive information
- [Alert - Expandable](./alert/alert-expandable) - Collapsible important information

- [Banner](./banner/) - Site-wide announcements
- [Loading indicator](./loading-indicator/) - System processing feedback
- [Modal](./modal/) - Focused interactions requiring user response
- [Snackbar](./snackbar/) - Brief feedback messages

### Navigation
Help users move through the site and understand their location.

- [Back to top](./back-to-top/) - Return to page top
- [Breadcrumbs](./breadcrumbs/) - Location within site hierarchy

## Deprecated components

The following components are deprecated and should not be used in new development:

- **Banner - Promo** - Use standard [Banner](./banner/) instead
- **Input message** - Use [Alert](./alert/) (slim variant) instead
- **Notification** - Use [Alert](./alert/) instead
