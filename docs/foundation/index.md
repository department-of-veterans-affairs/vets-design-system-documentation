# Foundation

Foundation includes the core design decisions that inform all components and patterns on VA.gov. These resources help maintain a consistent, accessible experience for Veterans.

## Core Elements

- **[Color Palette](./color-palette)** - Primary, semantic, and utility colors with accessibility guidance
- **[Typography](./typography)** - Font families, sizes, weights, and line heights
- **[Spacing Units](./spacing-units)** - Consistent spacing scale based on multiples of 8
- **[Breakpoints](./breakpoints)** - Responsive design breakpoints for mobile-first development
- **[Design Tokens](./design-tokens)** - Standardized design decisions expressed as tokens

## Layout

- **[Layout Overview](./layout/)** - Page structure and grid system fundamentals
- **[Flexbox Grid](./layout/flexbox-grid)** - 12-column responsive flexbox grid
- **[Page Layouts](./layout/page-layouts)** - Common page layout templates

## Assets

- **[Logos](./logos)** - VA logo usage guidelines
- **[Image Aspect Ratios](./image-aspect-ratios)** - Standard ratios for images and banners
