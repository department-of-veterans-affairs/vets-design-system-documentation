# Font style

The font style utility enables control over whether text displays in italic or normal styling.

**CSS Property:** `font-style`
**Responsive:** Yes

## Available classes

| Class | CSS Value | Effect |
|-------|-----------|--------|
| `.vads-u-font-style--italic` | `font-style: italic` | Renders text in italicized format |
| `.vads-u-font-style--normal` | `font-style: normal` | Renders text in standard, upright format |

## Usage

```html
<p class="vads-u-font-style--italic">
  This text is italicized.
</p>

<em class="vads-u-font-style--normal">
  This emphasized text is rendered normally.
</em>
```
