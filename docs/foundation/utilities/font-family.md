# Font family

The font family utility provides CSS classes for controlling typeface selection across applications.

**CSS Property:** `font-family`
**Responsive:** Yes

## Typefaces

| Typeface | Font | Usage |
|----------|------|-------|
| Serif | Bitter | Serif typography needs |
| Sans-serif | Source Sans Pro | Primary sans-serif (default) |
| Monospace | System monospace | Code blocks and technical content |

## Available classes

| Class | Typeface |
|-------|----------|
| `.vads-u-font-family--sans` | Source Sans Pro |
| `.vads-u-font-family--serif` | Bitter |
| `.vads-u-font-family--mono` | Monospace |

## Usage

```html
<p class="vads-u-font-family--serif">
  This text uses the Bitter serif typeface.
</p>

<code class="vads-u-font-family--mono">
  Code displayed in monospace
</code>
```
