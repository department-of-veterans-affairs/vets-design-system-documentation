# Text color

The text color utility modifies text and SVG element colors using the VA.gov color palette.

**CSS Property:** `color`
**Responsive:** Yes

:::warning Accessibility
Not all of these colors are placed in accessible color combinations. Ensure adequate contrast ratios when applying colors.
:::

## Base colors

| Class | Color |
|-------|-------|
| `.vads-u-color--base` | Base text color |
| `.vads-u-color--white` | White |
| `.vads-u-color--black` | Black |
| `.vads-u-color--orange` | Orange |
| `.vads-u-color--link-default` | Default link color |
| `.vads-u-color--warning-message` | Warning message color |
