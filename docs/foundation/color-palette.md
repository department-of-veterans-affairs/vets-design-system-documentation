# Color Palette

The VA.gov Design System color palette is built on the U.S. Web Design System (USWDS) color tokens, ensuring consistency with federal design standards while meeting VA-specific needs.

## Primary Colors

The primary color family is used for primary actions, links, and key UI elements.

| Token | Hex | Usage |
|-------|-----|-------|
| `vads-color-primary` | #005ea2 | Base primary blue |
| `vads-color-primary-dark` | #1a4480 | Hover states |
| `vads-color-primary-darker` | #162e51 | Active states |
| `vads-color-primary-light` | #73b3e7 | Light accents |
| `vads-color-primary-lighter` | #d9e8f6 | Backgrounds |

## Accessibility

- Ensure sufficient color contrast ratios (4.5:1 for normal text, 3:1 for large text)
- Never use color alone to convey meaning
- Test with color blindness simulators
- All semantic colors include lighter variants for backgrounds that maintain contrast with text
