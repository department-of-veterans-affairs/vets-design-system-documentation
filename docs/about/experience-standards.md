# Experience standards

The Experience Standards serve as evaluation criteria used by the VA's Governance team to assess products across VA.gov and the VA Mobile App for design, information architecture, content, and accessibility during the Collaboration Cycle, particularly at staging reviews.

## Purpose

These standards center on end-user experience for Veterans, families, caregivers, and the public. Teams can self-check their work against these criteria throughout development to verify alignment with user needs.

## Seven user experience categories

### 1. Comprehension

User easily understands the product. Launch-blocking issues occur when users struggle to understand task completion.

**Standards:**
- Element perception (users can perceive meaningful elements)
- Content best practices adherence
- Sufficient task information availability
- Clear button/link destination expectations

### 2. Consistency

User has a predictable and unchanging experience across all VA.gov and VA Mobile App services.

**Standards:**
- Accurate, consistent content
- Best-practice components and patterns
- Clear, consistent design components
- Content alignment with other VA.gov content
- Current design component versions
- Alignment with VA.gov design principles

### 3. Credibility

User perceives VA.gov and VA Mobile App as a credible source.

**Standards:**
- Voice and tone alignment with content principles
- Information review/edit capability before submission
- Appropriate status notifications

### 4. Efficiency

User accomplishes goals quickly with minimal cognitive load.

**Standards:**
- Primary task focus without distractions
- No unnecessary repetition
- User notification before flow interruptions

### 5. Findability

User identifies location within VA.gov and navigation next steps.

**Standards:**
- Clear next steps identification
- Site location awareness
- Accessible content pathways

### 6. Health and safety

User safely accesses tools without health risks.

### 7. Usability

User easily knows how to use the interface.

**Standards:**
- Flow completion capability
- Element identification and state recognition
- Predictable component interactions
