# Feedback

Multiple channels are available for submitting suggestions, reporting issues, and communicating with the design system team.

## Feedback channels

### Slack support

Get immediate support for technical issues, urgent matters, and quick questions through the #platform-design-system Slack channel.

### GitHub issues

Submit issues through the GitHub repository at `department-of-veterans-affairs/vets-design-system-documentation`.

### Email contact

For content style guide questions, email **vawebdesign@va.gov** (include "Content style guide" in the subject line).

## Feedback processes

### New components or patterns

Follow the dedicated [contributing process](./contributing) when suggesting additions to the design system.

### Updates to existing components

For minor changes, bugs, and documentation improvements, review the contribution guidance to ensure you follow the correct steps.

### Component-specific feedback

Each component includes a dedicated GitHub discussion for collecting usage feedback. Visit the component's documentation page to find the discussion link.

## What happens next

After submitting feedback:

1. The Design System Team reviews submissions
2. Issues are triaged and prioritized
3. Updates are communicated through the #platform-design-system Slack channel
4. Major changes are announced in the [What's new](./whats-new) section
