# Content Style Guide

The VA.gov Content Style Guide provides standards for creating clear, consistent, and accessible content for Veterans and their families. Following these guidelines helps VA speak with one voice across all digital properties.

## Core Principles

- Write in **plain language** that Veterans and their families can understand
- Use **conversational tone** that addresses users directly
- Follow **web best practices** for scannable, accessible content
- Maintain **consistency** across all VA administrations
