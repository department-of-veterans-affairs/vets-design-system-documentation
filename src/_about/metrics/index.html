Accessibility Issues by Defect Level

Tracks the number of new accessibility issues created each month, grouped by defect severity level (0 = most severe, 3 = least severe). Dashed vertical lines mark calendar year quarter boundaries.

This provides visibility into the number and severity of open accessibility issue tickets over time. It is a reporting signal, not a measure of overall system health, and should be interpreted alongside context such as testing activity and policy changes.
