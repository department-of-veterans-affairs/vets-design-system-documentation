# Ask users for...

Patterns for collecting user information through forms and applications. These patterns provide consistent, accessible approaches for gathering:

- Personal information (names, dates of birth)
- Contact details (addresses, phone numbers, email)
- Preferences and responses
- Files and documents

Browse the patterns in this section to find guidance for your specific data collection needs.
