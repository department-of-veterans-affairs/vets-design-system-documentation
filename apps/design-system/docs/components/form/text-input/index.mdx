    - value: Accepting currency amounts or numeric values
    - value: Fields where autocomplete functionality would enhance user experience
  when_to_consider_else:
    - value: For selecting from predefined options (use [Select](../select) or [Radio Button](../radio-button))
    - value: For multiple-line text responses (use [Textarea](../textarea))

  - value: Required indicators are communicated via text, not symbols alone
