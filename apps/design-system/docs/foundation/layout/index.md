# Layout

The layout foundation provides tools for building page structures on VA.gov, offering design flexibility through modern CSS specifications.

## Overview

The VA.gov Design System uses a **flexbox-based grid** that delivers more design flexibility and options for responsive design compared to traditional float-based grids.

## Key Components

### Flexbox Grid

A 12-column responsive grid system built on CSS flexbox. The grid provides:

- Responsive column widths
- Automatic equal-width columns
- Nested grid support
- Gutters via padding utilities

[Learn more about the Flexbox Grid →](./flexbox-grid)

### Page Layouts

Pre-built page layout templates for common VA.gov page types:

- Two columns with content on right
- Two columns with content on left
- Full-width layouts

[Learn more about Page Layouts →](./page-layouts)
