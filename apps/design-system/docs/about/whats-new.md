## Content style guide enhancements

### August 2025

**Dates and numbers:** Added exception for automated timestamps using 3-letter time zone format (for example, EDT).

**Links:** Emphasized placing links on separate lines; inline links require content team consultation.

**Numbers, signs, and symbols:** Clarified "one" versus "1" usage in specific expressions like one-to-one and one-time scenarios.

**Punctuation:** Allowed quotation marks for button/section references when text is already bold in error messages.

## Accessibility improvements

Focus management guidance updated with refined recommendations for better user experience across applications.
